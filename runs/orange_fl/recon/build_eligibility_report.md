# Orange County build eligibility report

`READY_WITH_BLOCKERS`

## Reason

Orange County has a verified official primary event source in the Comptroller Official Records search and an officially linked foreclosure-sales portal. Public court search, tax deed sales, and code-violation search add supporting or limited event coverage. The Property Appraiser and GIS sources are enrichment only. Parcel-level tax delinquency remains unresolved because the official Tax Collector material documents the process but does not expose a verified public list or bulk feed.
